# DevOps Bash Toolkit implementation

# Features
- User Info Collection
- System Monitoring
- File Management
- Backup Automation
- Process Monitoring 

# Setup

chmod +x scripts/*.sh
chmod +x run_all.sh

# Usage

./scripts/user_info.sh
./scripts/system_check.sh
./scripts/file_manager.sh create file.txt
./scripts/backup.sh /path/to/folder
