# DevOps Bash Toolkit Assessment

This project is a Bash-based DevOps toolkit for basic automation, monitoring, file handling, and backups.

## Features

- User information collection with validation
- System resource checks
- File management commands
- Directory backup with cleanup of old backups
- Optional process monitoring
- Optional interactive runner script
