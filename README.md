# DevOps Bash Toolkit

## 📌 Overview

This project is a DevOps Bash Toolkit that automates system monitoring, file management, backups, and process checks.

## 🚀 Features

* User information collection and validation
* System health monitoring (CPU, memory, disk)
* File management (create, delete, rename, list)
* Automated backups with cleanup
* Process monitoring (bonus)
* Interactive menu to run all scripts (bonus)

## 📁 Project Structure

```
.
├── scripts/
├── logs/
├── backups/
├── run_all.sh
├── README.md
```

## 🧑‍💻 How to Run

### Make scripts executable

```
chmod +x scripts/*.sh
```

### Run scripts

```
./scripts/user_info.sh
./scripts/system_check.sh
./scripts/file_manager.sh create file.txt
./scripts/backup.sh .
```

### Run all (bonus)

```
./run_all.sh
```

## 📊 Logs

All outputs are saved in the `logs/` directory.
